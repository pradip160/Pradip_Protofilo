## Projects Included

### IT Support Troubleshooting Knowledge Base

A beginner IT support knowledge base covering common first-line support issues such as slow laptops, Wi-Fi problems, printer issues, login problems, Outlook/Teams issues, VPN connection problems, software installation requests, and OneDrive sync issues.

### AI Mobile Application Security: Insecure Data Storage

A cybersecurity project focused on insecure data storage in mobile applications. The project explored how sensitive data such as passwords, tokens, and personal details can be exposed when stored insecurely. It also discussed secure storage methods such as Android Keystore, EncryptedSharedPreferences, encryption at rest, least privilege, and secure development practices.

### Machine Learning for Cyber Resilience of Hydrogen Energy Systems

A machine learning project focused on anomaly detection for cyber resilience in hydrogen energy systems. The project compared Logistic Regression, Random Forest, and Isolation Forest models using performance metrics such as accuracy, precision, recall, F1-score, ROC-AUC, and confusion matrix.

### NorthStar Urban Mobility and Logistics Data Analytics Project

A data analytics and NoSQL database design case study based on an urban mobility and logistics organisation. The project explored structured and semi-structured data and proposed a MongoDB Atlas document database approach.

### Java Application and Website Development Projects

Small Java applications and basic website development projects created to practise programming, object-oriented programming, debugging, logic building, and front-end development.

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/pradip160/Pradip_Protofilo.git
```

2. Open the project folder:

```bash
cd Pradip_Protofilo
```

3. Open `index.html` in a browser.

You can also use the Live Server extension in Visual Studio Code.

## How to Update the CV

Place the updated CV PDF inside the `assets` folder using this exact file name:

```text
Pradip_Mainali_CV.pdf
```

The Download CV button in the website is linked to:

```text

```

## How to Update the Profile Photo

Place the profile photo inside the `assets` folder using this exact file name:

```text

```

The image in the website is linked to:

```text

```

If the image is not available, the website can show a placeholder with initials.

## Deployment Using GitHub Pages

To publish the website:

1. Go to the GitHub repository.
2. Click **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, choose:

   * Source: **Deploy from a branch**
   * Branch: **main**
   * Folder: **/root**
5. Click **Save**.
6. Wait a few minutes for GitHub Pages to publish the website.

Live website format:

```text

```
